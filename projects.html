RISC V Out of Order Superscalar Processor

◼ Designed a parametrizable 4 issue out of order core using SystemVerilog, supporting all RV32I instructions
◼ Explicit register renaming scheme, non blocking 4 way associative L1 cache and 7 pipelined execution units

◼ Developed a random instruction stream generator in Python to stress test the out of order core
◼ Developed an automated regression testing script in Shell, synthesized with DC. The processor runs at 333 MHz under 130nm node
